Product surface

不是聊天窗口套壳。

Fellow、技能、模型、权限、云端房间和桌面桥接都在同一套 GUI 里。

Local-first 本机工具留在本机

Claude Code 和 Codex 继续使用你的 PATH、账号和文件系统。

Cloud rooms 协作状态可同步

DM、群聊、消息序列和 Fellow 提及由云端保持一致。

Fellow memory 身份和技能可管理

技能、桌宠、头像、记忆和模型配置围绕 Fellow 管理。

Trust boundary

远程协作可以轻，执行边界必须清楚。

Hermes packaged. Claude Code and Codex stay yours. 内置运行时负责默认体验；你自己的 CLI Agent 仍由本机环境和账号控制。
权限先问，再执行 用户能看到任务计划、命令意图和执行位置，而不是只看流式回复。
Bridge 在线才调用你的 Fellow 远程请求只有在 owner desktop 在线并绑定时才会进入本机执行路径。
